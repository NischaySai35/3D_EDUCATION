# 3D_EDUCATION

A modern, interactive learning web app that lets users open any 3D model, hover to highlight parts, and get AI-powered explanations of the selected components — perfect for teaching, demos, and hands-on learning.

![Website view](public/image.png)

![Demo Snapshot](public/vid.mp4)

Short description
- Explore real-time 3D models with smooth navigation, intuitive hover-highlighting, and contextual AI explanations for any part of the model.

Why this exists
- Visual learners gain immediate context by interacting with models.
- Teachers and creators can present complex structures with clickable, explainable parts.
- Useful for engineering, biology, architecture, product demos, and more.

Key Features
- Load and view 3D models (GLTF / GLB / OBJ — extendable).
- Hover-highlight model parts for easy identification.
- Click a highlighted region to request an AI-generated explanation of that part.
- Clean, minimal UI with a focused learning pane for explanations.
- Lightweight server: clone, install, and run — ready for local demos or small deployments.

Quick Start (local)
- Prerequisites: `Node.js` (v14+), `npm`.
- From project root:
```powershell
npm install
node [server.js](http://_vscodecontentref_/4)
```
